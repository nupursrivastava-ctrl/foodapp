
Food Photo Calorie Tracker — Web PWA (MVP)
This repo contains a Next.js frontend and FastAPI backend that process food photos using Hugging Face / Replicate models and map estimated portions to calories using USDA FoodData Central.

Quick local run:

Install dependencies: yarn
Start dev: yarn dev
Frontend at http://localhost:3000, backend at http://localhost:8000
See /docs/deploy.md for detailed deploy steps (Vercel / Render) and environment setup.
